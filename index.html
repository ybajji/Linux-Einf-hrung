What is Linux ?

Linux is an operating system like Windows and MacOS.

As you can see on the right, Linux looks similar to the other two systems.

So why do we need Linux at all ?

This is best explained by the following chain of information

  • is an open source operating system
  • Therefore there are no superfluous programs or applications ( Microsoft Edge Browser)
  • Therefore Linux runs faster than Windows and MacOS
  • That's why most websites and servers use Linux as operating system (almost 66% of the whole internet).
